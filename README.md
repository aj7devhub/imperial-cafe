# ☕ Imperial Cafe

## Overview

**Imperial Cafe** is a demo café website designed to showcase modern web development practices and UI/UX design concepts. It serves as a template and learning resource for developers who want to understand how a café or restaurant website can be structured and developed.

🌐 **Live Demo:** https://ajdevhub-cafe.netlify.app

Created by **aj7devhub** with valuable contributions from **KajalSingh-17**.

> Last Updated: January 2026

---

## About The Project

This project is not a production-ready application but rather a demonstration website that highlights the type of web experiences I can build. It acts as a foundation for future enhancements and can be used as a reference for designing café, restaurant, or food-service websites.

## Current Features

At its current stage, Imperial Cafe provides:

* A complete website layout and design structure
* Modern and responsive UI
* Menu and café-themed pages
* Demonstration of frontend and backend project architecture

### Current Limitations

* The website currently functions as a visual prototype.
* Buttons and actions are not yet interactive.
* No authentication or payment functionality has been implemented.
